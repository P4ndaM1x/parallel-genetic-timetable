 This repository contains a C++ program that solves the School Timetabling Problem using a genetic algorithm parallelized with MPI (Message Passing Interface).\hypertarget{index_autotoc_md1}{}\doxysection{Quickstart}\label{index_autotoc_md1}
\begin{quote}
\mbox{[}!\+NOTE\mbox{]} All the following commands should be executed from the root of the project. \end{quote}
Since in-\/source building is forbidden, creating build directory for CMake is mandatory\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{mkdir ./build}

\end{DoxyCode}


Then, the {\ttfamily main} binary can be easly built\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{cmake -\/B ./build -\/S ./ \&\& cmake -\/-\/build ./build}

\end{DoxyCode}


And run\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{./build/main}

\end{DoxyCode}
\hypertarget{index_autotoc_md2}{}\doxysection{Custom targets}\label{index_autotoc_md2}
Currently supported custom commands\+:
\begin{DoxyItemize}
\item {\ttfamily clear} -\/ removes whole {\ttfamily ./build} directory and creates it once again, without any content
\item {\ttfamily test} -\/ runs the algorithm with sample data
\end{DoxyItemize}

Example usage\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{cmake -\/-\/build ./build -\/-\/target clear}

\end{DoxyCode}
\hypertarget{index_autotoc_md3}{}\doxysection{Parallel environment}\label{index_autotoc_md3}
To unleash the parallel capability, firstly setup the environment, e.\+g.\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{. ./setup.sh}

\end{DoxyCode}


Secondly, execute binary with {\ttfamily mpiexec}, e.\+g. using provided script\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{./run.sh <custom-\/parameters>}

\end{DoxyCode}
